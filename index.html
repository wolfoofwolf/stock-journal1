🚀 自訂你的專屬網址

只需 3 步驟，讓你的回測平台擁有專業網址！

1
建立 GitHub 倉庫
建議倉庫名稱：stock-journal
2
上傳 index.html

3
啟用 GitHub Pages
Settings → Pages → Source: main → Save

https://你的帳號.github.io/stock-journal
